Hello, welcome to my page!
In this page, I am putting on a play from the movie Titanic, which will be displayed on a tablet, that the actors can touch the name of the character they are playing so they can highlight their lines to make it easier to rehearse. This single page application is done using the DOM and javascript. The page will do the work without page re-loads.

I created a function which passes in an element by reference, which is a live instance of the element on the page, and changes the background color. I added the capability of toggling, switching between 2 states, the background color of the element passed into a function. I have also built an anonymous function into the event listener loop. A new function is created in which I pass in the identity of the current actor (ROSE,RUTH, MOLLY, and JACK) to the function so that when clicked, the element always knows its name.

THis page works with a single click; whenever the actors click on the name/ line of thier character, then all lines for that specific actor in a play is highlighted with a dark bule color while the lines for the rest of the actors are highlighed with white making it easy for the actor to focus only on his or her lines.

Feel free to try it youself by clicking on any character's line of play!
